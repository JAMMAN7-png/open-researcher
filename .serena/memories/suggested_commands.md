# Suggested Commands

## Development
```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Environment Variables
Required in `.env.local`:
- `ANTHROPIC_API_KEY` - Anthropic API key
- `FIRECRAWL_API_KEY` - Firecrawl API key (can also be provided via UI)

## Linting/Formatting
```bash
# Run ESLint
pnpm lint
```

## System Commands (Windows)
- `dir` instead of `ls`
- `type` instead of `cat`
- PowerShell or Git Bash recommended for Unix-like commands
